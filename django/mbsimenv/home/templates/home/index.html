The MBSim Environment

Introduction

The MBSim Environment provides tools for simulation, visualization and analysis of multibody systems.

  • Graphical model design
  • Parametric model setup
  • Generate arbitrary user functions by use of symbolic expressions
  • Motion view in 3D
  • Output data plotting
  • Modules for control, hydraulics, physics and flexible bodies
  • Model exchange via Functional Mock-up Interface (FMI)
  • Full extensibility via C++ or Python interface

History

The history of MBSim and its accompanying tools goes back to the year 2007. It is continuously improved and extended since then.

2007
Start of the MBSim project {% octicon "link-external" %} at the Institute of Applied Mechanics {% octicon "link-external" %} at the Technical University of Munich {% octicon "link-external" %}. Initial commit of MBSim and fmatvec as internal subversion repositories.
2008
Birth of HDF5Serie as an internal subversion repository. Switching of the MBSim output files from ASCII to HDF5.
2009
Release of MBSim, HDF5Serie and fmatvec as an open source project on the hosting platform BerliOS {% octicon "link-external" %}.
2009
Birth of OpenMBV {% octicon "link-external" %} as an internal subversion repository as an improved and modernized successor of AMVis {% octicon "link-external" %}.
2009
Birth of h5plotserie, the GUI for time-series post-processing of MBSim.
2009
Birth of MBSimXML, the XML based model format of MBSim.
2011
The hosting platform BerliOS {% octicon "link-external" %} closes. Move of MBSim, OpenMBV, HDF5Serie and fmatvec to the hosting platform Google-Code {% octicon "link-external" %}.
2012
Birth of MBSimGUI, the GUI of MBSim.
2015
The hosting platform Google-Code {% octicon "link-external" %} closes in 2016. Birth of the GitHub {% octicon "link-external" %} organization mbsim-env {% octicon "link-external" %}. Move of MBSim, OpenMBV, HDF5Serie and fmatvec to the hosting platform GitHub {% octicon "link-external" %}.
2016
Birth of the continous integration (CI) and nightly build system of MBSim-Env on a commercial vServer.
2019
Switched the CI and nightly build system to Docker {% octicon "link-external" %} containers and switched the vServer hosting provider.
2020
Redesign of the CI and nightly build system based on Django {% octicon "link-external" %}.
